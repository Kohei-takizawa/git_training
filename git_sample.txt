Gitの練習です

・適当なフォルダで右クリックしてgit bashを開く

・git init

・git status

・git add git_sample.txt

・git status

・git commit -m "training version 1.0"

・git status

・git log

・GitHubにログインして「Create repository」ボタン押下してリモートレポジトリを作成する

・git remote add origin git@github.com:Kohei-takizawa/git_training.git

・git push origin main

・ここまででGitHub上のgit_trainingのリモートレポジトリが作成された

・別のフォルダで2つ目のgit bashを開く

・git clone git@github.com:Kohei-takizawa/git_training.git

・最初に開いたgit bashでtxtファイルを修正してadd, commit, push

・git pull
